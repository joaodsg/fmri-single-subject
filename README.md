# fmri-single-subject
**Dataset:** Single-subject preprocessed resting-state fMRI file `sub-17017_task-rest_bold.nii.gz` from OpenNeuro **ds004564 — BIDS Resting State fMRI Dataset**.  

**Subjects:** sub-17017, sub-19738  
**Scanner:** Siemens 3T, TR=2.0s, TE=30ms, Flip angle=90°, 32-channel head coil, 3mm isotropic resolution  
**Preprocessing:** Motion correction, slice-timing correction, spatial smoothing (6mm FWHM), high-pass filtering (0.01 Hz), using FSL 6.0.4  

**Note:** The fMRI file is **not included** in this repository due to size limits.
